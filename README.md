# Stamp

#### **직원 출결관리 서비스**



## 프로젝트 소개

#### 직원들의 출결을 파악하고 관리할 수 있는 서비스입니다.

- 직원관리, 부서관리, 직원 출결 목록을 확인할 수 있는 **관리자 페이지**
- 출결 체크 및 내 정보를 수정할 수 있는 **직원용 페이지**

<br/>
